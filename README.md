# js-mentor-npm

[Brief description of the package -- what does it do?]

This is a utility for parsing files and providing metrics about the file.

## Quick start

```bash
$ npm -g i js-mentor-npm
# install the program
$ js-mentor-npm -u -i foo.txt
# Parses foo.txt
```

## Options

| Option | Description               |
| ------ | ------------------------- |
| -u     | Unique word count         |
| -i     | Path to the file to parse |
